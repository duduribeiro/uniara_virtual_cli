# UniaraVirtualCli

A basic gem to retrieve grades from uniara virtual

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'uniara_virtual_cli'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install uniara_virtual_cli

## Usage

### Retrieve Grades

```
uniara_virtual grades
```
